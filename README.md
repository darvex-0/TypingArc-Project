# TypingArc - Improve Your Typing Speed

TypingArc (TypingHub) is a modern, interactive typing speed test application built with vanilla HTML, CSS, and JavaScript. Test your typing skills, track your progress, and improve your words-per-minute (WPM) in a visually appealing environment.

## Features

- **Real-time WPM Calculation**: Accurately measures your typing speed in words per minute
- **Accuracy Tracking**: Monitors your typing accuracy percentage
- **Dark & Light Themes**: Toggle between dark and light modes for comfortable testing
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Modern UI**: Clean, professional interface with smooth animations
- **Statistics**: View your results and typing metrics
- **Instant Feedback**: Get immediate feedback on your typing performance

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation or backend required

### Installation & Usage

1. Clone or download this repository
2. Open the `index.html` file in your web browser
3. Click "Start Typing" to begin the test
4. Type the text displayed on screen as quickly and accurately as possible
5. View your results and statistics when complete

## How It Works

1. A passage of text is displayed on the screen
2. Start typing in the input field to begin the test
3. Your WPM, accuracy, and time elapsed are calculated in real-time
4. Complete the passage to see your final results

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with CSS variables, Flexbox, and Grid
- **JavaScript**: Vanilla JS for interactivity and calculations
- **Google Fonts**: Inter font family for clean typography

## Tips to Improve Your Typing Speed

- Focus on accuracy first, speed will follow
- Use proper finger positioning on the keyboard
- Practice regularly to build muscle memory
- Take breaks between sessions to avoid fatigue
- Challenge yourself with different difficulty levels
